# MedTravel China

Dental tourism platform connecting international patients with top dental clinics in China. Built with Next.js.

🌐 **Live**: [medtravel.jytech.us](https://medtravel.jytech.us/en)

## Features

- Multi-language support (EN/ZH)
- Clinic search and comparison
- Treatment cost calculator
- MedChat — AI-powered consultation before you fly
- Appointment booking system

## Tech Stack

- **Framework**: Next.js 16 / React 19
- **Styling**: Tailwind CSS 4
- **Language**: TypeScript
- **Deployment**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
